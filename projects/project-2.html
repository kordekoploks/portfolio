Project Summary

Active Health is a connected healthcare platform that enables users to monitor physiological data in real time through Bluetooth-enabled IoT devices. Built using Flutter for Android and iOS, the application securely collects health metrics, synchronizes data through cloud services, and provides users with timely notifications and actionable insights. Designed with scalability in mind, the platform combines cross-platform mobile development, Bluetooth connectivity, cloud infrastructure, and modern application architecture to deliver a reliable digital healthcare experience.

Designed for reliability and scalability, the platform combines Bluetooth communication, cloud synchronization, and an intuitive mobile experience to help users monitor their health anytime and anywhere.
